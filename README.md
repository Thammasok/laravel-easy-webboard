# Laravel Easy Webboard

## Install Composer
``` https://getcomposer.org/ ```

## Composer Install
``` composer install --no-dev ```
## Generate Key
``` php artisan key:generate ```

## Create Table
``` php artisan migrate ```

## Run Seeder
``` php artisan db:seed ```

## Local Development Server
``` php artisan serve ```

This command will start a development server at http://localhost:8000:
